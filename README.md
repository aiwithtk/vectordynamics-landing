# Vector Dynamics - Landing Page

Static landing page for Vector Dynamics, built using the Cream Editorial design system.

## Features
- Minimalist serif typography
- Custom YouTube explainer video
- Lead Magnets integration (Gumroad, Beehiiv, Skool)

## Deployment
This project is connected to Vercel and deployed automatically on every push to the `main` branch.
- **Production URL**: [https://vectordynamics-nine.vercel.app](https://vectordynamics-nine.vercel.app)
